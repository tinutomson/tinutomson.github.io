Study Genie

Study Genie is an application that helps students to create, collaborate and share notes. An elastic search API was set up to ease the search. Further, notes were classified and clustered based on content. This model is then used to provide content based recommendations to users. Further, similar users were also identified and recommendations were tuned according to collaborative filtering. Application also allowed users to create print friendly cheatsheets out of notes using the application.
